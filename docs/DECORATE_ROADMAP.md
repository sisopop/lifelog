# 다꾸(다이어리 꾸미기) 설계 로드맵

> MZ세대 타겟 출시의 핵심 차별화 = 일기장 꾸미기.
> 작성/갱신: 2026-06-20. 현재 구현: 표지 **색상** + **아이콘**(commit 87fabad, 78fdd2d).

## 핵심 설계 원칙 — "조합형 레이어"

노트를 **하나의 통짜 디자인**이 아니라 **독립 레이어의 조합**으로 모델링한다.
같은 리소스를 조합해 수백~수천 가지 스타일을 만들 수 있어, 꾸미기샵 콘텐츠를 계속 확장하기 쉽다.

예시 조합:
| 레이어 | 값 |
|--------|-----|
| 노트 구조(제본) | 왼쪽 스프링 |
| 표지 재질 | 갈색 가죽 |
| 잠금 | 고무 밴드 |
| 아이콘 | 🌲 |
| 속지 | 크림색 도트 |
| 페이지 프레임 | 빈티지 |

→ **"노트 구조"와 "속지"는 반드시 분리.** (구조=겉, 속지=안)

## 데이터 모델 방향 (제안)

`Journal`에 흩뿌리지 말고 `JournalStyle` 값 객체로 묶는다(직렬화/마이그레이션/꾸미기샵 적용 용이).

```
JournalStyle {
  int coverColor;        // ✅ 구현됨
  String? icon;          // ✅ 구현됨
  String? pattern;       // 패턴(점/스트라이프/체크…)
  String binding;        // 제본: leftSpring/topSpring/hardcover/leather…
  String material;       // 표지 재질: solid/leather/linen/kraft/denim…
  String? lock;          // 잠금: band/strap/buckle/ribbon…
  String? clip;          // 클립/장식: bookmark/corner/nameplate…
  String corner;         // 모서리: square/round/metal/worn
  String pageStyle;      // 속지: ruled/grid/dot/blank/cream/vintage…
  String? pageFrame;     // 페이지 프레임: floral/seasonal/vintage…
  String? bookmark;      // 북마크: ribbon/tab/index…
  String? themePack;     // 특수/시즌 테마 묶음 id
}
```

마이그레이션: 기존 `coverColor`/`icon` 유지, 신규 필드는 기본값 백필(solid/square/ruled 등).

## 추가 아이디어 / 제안 (Claude)

1. **테마 = 레이어 조합 프리셋.** 프리미엄 "여행 테마"는 새 엔진이 아니라
   `{binding:leather, material:vintage, lock:strap, pageStyle:kraft, frame:vintage}` 같은
   기존 레이어 조합으로 정의 → 콘텐츠 추가 비용 거의 0.
2. **v1은 전부 절차적(CustomPainter/그라데이션/SVG).** 비트맵 에셋 없이 패턴·질감·제본·잠금까지
   코드로 그려 출시 → 용량↓, 디자이너 의존↓. 일러스트 팩만 후속.
3. **미리보기 일관성.** 현재 시트의 미니 표지 미리보기를 공용 `JournalCover` 위젯으로 추출해
   책장/시트/꾸미기샵/상세 헤더가 같은 렌더러를 쓰게 → 레이어 추가 시 한 곳만 수정.
4. **무료로 충분히 예쁘게 → 후킹.** 무료(색·패턴·아이콘·모서리)만으로도 만족스럽게 만들고,
   프리미엄은 "재질감·제본·테마"로 차별. 첫 진입 장벽을 낮춰 리텐션 확보.
5. **시즌 자동 추천.** 날짜 기반으로 "벚꽃(봄)·여름 바다·핼러윈·크리스마스" 테마를 시즌에 노출 → 재방문 유도.
6. **'잠금'은 시각 장식 + 실제 기능 연계 가능.** 자물쇠/버클 스타일은 나중에 앱 잠금(PIN/생체)과 묶어
   "비밀일기" 특수 스타일로 → 꾸미기가 기능과 연결되는 스토리.
7. **표지 미리보기를 공유 카드로.** 꾸민 일기장 표지를 이미지로 내보내 SNS 공유 → 자연 바이럴(MZ).
8. **접근성/가독성 가드.** 표지 색 대비, 어두운 표지일 때 제목 텍스트 색 자동 반전 규칙 1개는 넣어두기.
